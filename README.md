# What is this?
This is a special repository for my homelab designed to mimic a realistic corporate environment that has today's standards in terms of security. Here can be found all documentation of my discoveries, what I change, documentations of malware ran with the intent to test its impact, and attempts at unauthorized access.

## Setup
### 1. Worker
Operating System (OS): Windows 11 Professional\
The worker is connected to an Active Directory domain for work purposes.
### 2. Threat Actor
Operating System (OS): Kali Linux\
The threat actor will be used to test vulnerabilities and their impact. This one will rarely be used due to being very specific regarding purpose.
### 3. Domain Controller
Operating System (OS): Windows 11 - Windows Server 2025
This acts as the Active Directory Domain Controller and the server.
